## 🧠 Sobre o Projeto

Portfólio pessoal desenvolvido do zero com **HTML5, CSS3 e JavaScript Vanilla**, sem frameworks — desafiando a si mesmo a construir uma experiência rica e interativa com fundamentos puros da web.

Destaques técnicos:
- **Canvas API** para animação de partículas em tempo real
- **IntersectionObserver** para animações ativadas por scroll
- **EmailJS** para envio de formulário sem backend
- **Cursor personalizado** com interatividade via JS puro
- **Glassmorphism + gradientes** para UI estilo tech premium
